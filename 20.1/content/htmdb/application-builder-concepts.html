3 应用构建器 概念

要有效地使用 App Builder ，开发人员必须了解一些关键概念，包括如何管理用户界面设计，如何查看呈现的应用程序页面，了解页面处理和呈现，管理会话状态，理解URL格式语法，管理会话状态值，以及使用内置的替换字符串等。

  • 了解 Oracle Application Express Applications
    了解Oracle Application Express applications, 理解 数据库和Websheet应用程序之间的差异以及使用“创建应用程序向导”的优点。
  • 运行一个应用或页面
    要查看应用程序的最终被呈现的样子，需要把它交给Application Express引擎来运行。Application Express引擎根据存储在数据库表中的页面定义数据动态呈现和处理页面。
  • 页面呈现和处理是如何工作的?
    了解Application Express引擎如何呈现和处理页面，何时Oracle Application Express发出隐式提交，条件式呈现和处理如何工作以及如何验证用户或控制对控件和组件的访问。
  • 关于启用对书签的支持
    开发人员可以通过使用0作为会话ID或启用“重新加入会话”来协助那些把应用页面添加到浏览器书签的用户。
  • 了解会话状态管理
    Oracle Application Express透明地维护会话状态，并使开发人员能够从应用程序的任何页面获取和设置会话状态值。
  • 显示会话状态
    通过单击运行时开发人员工具栏上的 会话 来查看“会话”页面。
  • 了解URL格式
    Oracle Application Express 支持两类URL格式: 友好式 和 f?p 格式.
  • 管理会话状态值
    当用户在不同的应用程序页面之间导航时，管理会话状态以存储和检索用户的值。
  • 使用替代字符串
    在页面模板或区域源中使用替换字符串来传递信息并将字符串替换为另一个值。
