23.7 About Uploading Static Files to Your Web Server

To improve performance you should place static files on the Web server.

As a best practice, you should also enable gzip compression on the Web server so that the file size is minimized when transmitting the static files to the browser. Furthermore, you should also enable file caching for browsers by sending expiration headers.
