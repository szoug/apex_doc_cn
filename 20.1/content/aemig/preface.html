Preface

Oracle Application Express Application Migration Guide describes how to convert applications from Oracle Forms to Oracle Application Express.

  • Audience
  • Documentation Accessibility
  • Related Documents
  • Conventions

Audience

Oracle Application Express Migration Guide is intended for application developers who are building database-centric web applications using Oracle Application Express. This guide describes how to upload and annotate Oracle Forms applications when migrating to Oracle Application Express.

To use this guide, you must have a general understanding of relational database concepts and an understanding of the operating system environment under which you are running Oracle Application Express.
