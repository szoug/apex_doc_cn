13.3.6 Using Template Options

Template options provide a declarative way for developers to apply different styles to components on an Oracle Application Express page.

  • Managing Global Template Option Groups
    Developers can define the purpose of related template options by creating template option groups. Examples of template option groups include Button Size, Button Style, List Examples, List Style, and Form Label Position.
  • Managing Global Template Options
    Gobal template options are defined at the theme-level.
  • Managing Template Options
    Template options enable developers to declaratively apply CSS modifiers to the templates they have chosen for pages, regions, reports, lists, breadcrumbs, items and buttons.
