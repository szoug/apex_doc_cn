1 Changes in Release 20.1 for Oracle Application Express SQL Workshop Guide

All content in Oracle Application Express SQL Workshop Guide has been updated to reflect release 20.1 functionality and user interface changes.

New Features and Updates

The following are features and updates in Oracle Application Express release 20.1:

  • SODA Collections

    SQL Workshop now supports Simple Oracle Document Access (SODA). Use Object Browser to import JSON and ZIP archives into your workspace.

    参见 Managing SODA Collections.

  • Sample Queries Tab

    View sample code for querying tables and running commands in SQL Commands with the new Sample Queries tab in Object Browser.

    参见 Viewing Sample Queries for a Table.

  • Load Data from Cloud (Autonomous Data Warehouse only)

    Connect applications and workspaces to Cloud-based databases using Object Store References. Use Data Workshop to store Cloud credentials, then create external tables from Avro, Parquet, and other table types.

    参见 Loading Data From the Cloud (Autonomous Data Warehouse Only).
