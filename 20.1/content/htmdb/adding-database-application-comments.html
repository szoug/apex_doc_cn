7.12 Adding Database Application Comments

Use Application Comments to describe an application or track developers involved in the application development.

To create an application comment:

  1. Navigate to the Workspace home page.
  2. Click the App Builder icon.
  3. Select an application.
  4. On the Application home page, click Shared Components.

    The Shared Components page appears.

  5. Under Tasks, click Edit Application Comment.

    The Application Comments page appears.

  6. Enter comments in the Comments field and click Apply Changes.
