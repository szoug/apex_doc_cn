10.5.1 Example Faceted Search

View an example faceted search page.

A faceted search page features a faceted Search region and report. The Search region displays on the left side of the page and enables users to narrow down the search result by selecting facet values. Narrowing the search result, makes it easier for users to find the data they want. The right side of the page features a Search Results region which can display as a classic report or cards view.

The following shows a faceted search page built on the EBA_PROJECT_TASKS table. In this example the Project, Develop Bug Application, is selected and the report displays 11 rows.
