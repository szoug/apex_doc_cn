5.7.5 Reassigning the Issue Owner

Users with administrator privileges can reassign the issue owner.

To reassign the issue owner:

  1. From the Team Development menu, select Utilities.
  2. Click Reassign Owner.

    The Reassign Owner page appears.

  3. On Reassign Owner:
    1. Change From - Select current owner that you wish to reassign. Users marked as Orphan are current assignees whose related Application Express user has been removed.
    2. Change To - Select the new owner.
    3. For Open Issues Only? - Choose whether assignments should be changed for Open issues only, or for all issues.
  4. Click Next.
    A report appears detailing the issues to be reassigned.
  5. Click Change Owner.
