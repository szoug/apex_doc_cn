3 Using Oracle Application Express with Assistive Technologies and Keyboard

Users can access Oracle Application Express using only a keyboard or a screen reader such Freedom Scientific's screen reader JAWS.

Note:

JAWS with all the default settings was used in writing this section. All instructions in this section are based on using a full size keyboard. Note that instructions may vary if using a laptop keyboard.
