16.37 SET_NULL Procedure

This procedure sets procedures to set a DML column value to NULL. Useful when the row is initialized from a query context with set_values and the new value of one of the columns should be NULL.

Syntax

Signature 1

PROCEDURE SET_NULL(
    p_context               IN t_context,
    p_column_position       IN PLS_INTEGER );

Signature 2


PROCEDURE SET_NULL(
    p_context               IN t_context,
    p_column_name           IN VARCHAR2 );

Parameters

Table 16-35 SET_NULL Procedure Parameters

Parameter Description

p_context

Context object obtained with one of the OPEN_ functions.

p_column_position

Position of the column to set the value for within the DML context.

p_column_name

Name of the column to set the value.

Examples

Example 1
apex_exec.set_null(
     p_context         => l_dml_context,
     p_column_position => 6 );
Example 2
apex_exec.set_null(
     p_context         => l_dml_context,
     p_column_name     => 'SAL' );
