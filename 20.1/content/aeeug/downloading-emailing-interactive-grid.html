3.9 Downloading or Emailing an Interactive Grid

Obtain the data of an interactive grid as a CSV, HTML, or PDF file by selecting Download in the Actions menu.

To download an interactive grid:

  1. Click the Actions menu and select Download.
    The Download dialog displays.
  2. Select CSV, HTML, or PDF.
  3. For PDF, choose the Page Size, Page Orientation, and whether to include Accessibility Tags.
    Enabling Accessibility Tags enables the document to be read aloud by a screen reader and other text-to-speech tools (this increases the filesize of the PDF).
  4. Do one of the following:

    • To download the file, click Download.

    • To email the file, select the check box Send as Email. Perform the following steps:

      1. Complete the fields for the recipients (Address, Subject, Message).

      2. Click Send as Email. The app sends the file as an attachment to the email addresses.
