2 快速入门

"快速入门" 提供了Oracle Application Express的快速介绍。 本节说明如何访问您的开发环境，介绍工作区主页， 应用构建器(App Builder)主页和应用程序主页，并说明如何编辑帐户配置文件和用户首选项，以及如何访问产品文档。

Tip:

如果是访问本地安装环境，请先完成安装过程。 参见 Oracle Application Express 安装指南.

  • 关于访问你的开发环境
    登录和访问Oracle Application Express的方式取决于您的用户角色以及Oracle Application Express所在的位置。
  • 了解 Oracle Application Express
    Oracle Application Express 是一个集中托管/安装的声明性开发环境，用于开发和部署以数据库为中心的Web应用程序。
  • 了解工作区主页
    登录Oracle Application Express后，将显示工作区主页。 使用工作区主页访问关键的开发组件，包括 应用构建器(App Builder)， SQL 工作室(SQL Workshop)， 小组开发 和应用程序库。
  • 了解应用构建器(App Builder)
    开发人员使用 App Builder 创建和管理应用程序和应用程序页面。
  • 更改个人资料或密码
    编辑您的帐户个人资料以更新您的个人资料（电子邮件地址，名字或姓氏，上传个人资料照片或更改密码。
  • 编辑用户首选项并启用暗黑模式
    单击“帐户”菜单以启用或禁用暗黑模式或编辑用户首选项。
  • 使用Oracle Application Express文档
    Oracle Application Express提供了一个托管的在线文档库和字段级帮助。 您可以转到Oracle帮助中心的Oracle Application Express文档页面来浏览或搜索内容或下载整个库。
