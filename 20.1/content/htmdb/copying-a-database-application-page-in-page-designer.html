9.7 Copying a Database Application Page

You can copy a page from the current application or from another application. During the copy process, you can also copy shared components or change mappings to shared components in the target application.

To copy a page:

  1. View the page you want to copy in Page Designer:
    1. On the Workspace home page, click the App Builder icon.
    2. Select an application.
    3. Select a page.
    Page Designer appears.
  2. On the Page Designer toolbar, click the Create menu and then select Page as Copy.

    Description of the illustration pd_copy_pg.png
  3. On Copy Page Option, select one of the following and click Next:

    • Page in this application

    • Page in another application

  4. Follow the on-screen instructions.
