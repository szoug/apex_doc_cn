3 APEX_APPLICATION

The APEX_APPLICATION package is a PL/SQL package that implements the Oracle Application Express rendering engine. You can use this package to take advantage of many global variables. “Global Variables Available in APEX_APPLICATION” describes the global variables available in the APEX_APPLICATION package.
