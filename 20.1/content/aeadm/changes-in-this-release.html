1 Changes in Release 20.1 for Oracle Application Express Administration Guide

All content in Oracle Application Express Administration Guide has been updated to reflect release 20.1 functionality.

New Features and Updates

The following features are new in Oracle Application Express release 20.1:

  • Workspace Session Timeout Attributes

    Configure Session Timeout attributes at the workspace-level to reduce exposure at the application-level for abandoned computers with an open web browser.

    See "Configuring Session Timeout for a Workspace" and "Configuring Session Timeout for an Instance."

  • Data Dictionary Cache Report

    The Data Dictionary Cache report captures statistics and data analysis of all workspace schema tables and caches into the Oracle Application Express Data Dictionary. Manually refresh the Data Dictionary Cache manually from the Workspace Cache report.

    See "Viewing or Refreshing the Data Dictionary Cache."

  • Service Request History

    View the Service Request History report prior to requesting additional storage to determine whether prior storage requests have been declined due to some underlying error.

    See "Viewing the Service Request History for the Workspace."

Deprecated and Desupported Features

Legacy Team Development has been desupported in this release. All references to Legacy Team Development have been removed.

See "Deprecated Features" and "Desupported Features" in Oracle Application Express Release Notes.
