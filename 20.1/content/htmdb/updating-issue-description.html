5.2.4.3 Updating the Issue Description

View an issue and update the issue description (also known as the Issue Text).

Only an issue owner or an administrator can update an issue description (or text).

To update the Issue Text:

  1. View the issue:
    1. Click the Team Development menu.
    2. Click the issue title.

      The Issue Details page appears. You can update an Issue Text by either selecting Edit Issue Text from the Actions menu, or by clicking Show Actions, Edit.

  2. From the Actions menu, select Edit Issue Text.

    Description of the illustration edit_issue_text.png
  3. In the Edit Issue dialog:
    1. Update the description of the issue. Use the Markdown tools above the field to add formatting.
    2. Click Preview to preview the HTML.
    3. Click Apply Changes.
