3.4 关于启用对书签的支持

Developers can assist users who bookmark application pages by either using zero as the session ID, or enabling Rejoin Sessions.

In previous releases, if the pages within an application were public and did not require authentication, you could assist users in bookmarking pages by using zero as the session ID. Although this approach is still supported, Oracle Application Express now includes a new feature called Rejoin Sessions that enables developers to control this type of support at the page, application, or instance-level.
