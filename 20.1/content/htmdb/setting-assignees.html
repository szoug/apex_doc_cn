5.2.4.5 Setting Assignees

View an issue and then set assignees.

Assignees are users in the Oracle Application Express workspace. To create new assignee, a workspace administrator or Instance administrator must create new user.

To set assignees:

  1. View the issue:
    1. Click the Team Development menu.
    2. Click the issue title.

      The Issue Details page appears.


      Description of the illustration issue_details_action_menu.png
  2. From the Actions menu, select Set Assignees.

    You can also click the Set Assignees icon on far right side of the page.

  3. In the Assignees dialog:
    1. Select assignees to add to the issue.
    2. Click Apply Changes.

    A new event is added to the Event Timeline and the new assignees display under Assignees in the far right region.
