A.1.3 About List of Values Implementation

Oracle Forms uses record groups to define the query for list of values. Oracle Application Express dynamic list of values (LOVs) enable one display value and one return value to be defined within a SQL query. You can utilize the SQL query from the record group as the basis for the list of values query in Application Express.You can specify LOVs individually for an item or define as a named shared component which can be used with any item throughout the application.
