E.4 About Images Displaying Incorrectly in Oracle Application Express

Learn about troubleshooting if images in Oracle Application Express do not display correctly.

If images in Oracle Application Express do not display correctly, you may have more than one definition of the /i/ alias. To address this issue:

  • If possible, rename the first instance of /i/ to a different alias name.

  • Alternatively, copy the images from the directory where Oracle Application Express was downloaded or the images copied for Oracle REST Data Services (ORDS) to the directory defined by the first /i/ alias.
