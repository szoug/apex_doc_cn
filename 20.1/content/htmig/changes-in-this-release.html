1 Changes in Release 20.1 for Oracle Application Express 安装指南

All content in Oracle Application Express 安装指南 has been updated to reflect release 20.1 functionality.

Deprecated and Desupported Features

Deprecated features include:

  • mod_plsql is deprecated as of Oracle HTTP Server 12c (12.1.3). For more information, see My Oracle Support Note 1576588.1. Oracle recommends using Oracle REST Data Services.

  • Embedded PL/SQL gateway is deprecated and will be desupported in a future release. Oracle recommends using Oracle REST Data Services.

See "Deprecated Features" and "Desupported Features" Oracle Application Express Release Notes.
