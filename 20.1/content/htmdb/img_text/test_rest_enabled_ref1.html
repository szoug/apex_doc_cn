Description of the illustration test_rest_enabled_ref1.png

This illustration shows the Test REST Enabled SQL Service message that appears if you create the REST Enabled SQL service reference correctly. The example shown in this illustration reads: Success. The REST Enabled SQL Service is available. The test query returned the following result. Hello, Application Express! You are connected as SCOTT_OBE. The local database timestamp is: 2018-03-28 16:26:59 -07:00.
