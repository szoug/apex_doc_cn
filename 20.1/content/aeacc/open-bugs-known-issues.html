5 Open Bugs and Known Issues

Review known issues in Oracle Application Express in the following documents.

VPAT

VPATs for Oracle Database related products - VPAT for Oracle Application Express.

Click the link, click Database Products or Open All, then find the relevant version of Application Express.

The VPAT lists exceptions in the product, along with their corresponding bug number. Bugs are categorized as one of the following:

  • Affecting the Development Environment

  • Affecting Websheet Applications

  • Affecting Database Applications

  • Affecting Documentation
