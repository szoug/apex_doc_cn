5.2.4.2 Updating the Issue Title

View an issue and then update the issue title.

Only an issue owner or an administrator can update an issue title.

To update the issue title:

  1. View the issue:
    1. Click the Team Development menu.
    2. Click the issue title.

      The Issue Details page appears.

  2. From the Actions menu, select Edit Title.

    Description of the illustration issue_edit_title.png
  3. In Edit Issue Title dialog:
    1. Issue Title - Update the title.
    2. Click Apply Changes.
