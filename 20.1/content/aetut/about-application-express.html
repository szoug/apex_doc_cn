1 About Oracle Application Express

Oracle Application Express is an environment designed for rapid application development and deployment.

You can develop professional applications that are both fast and secure using only a web browser and limited programming experience. Oracle Application Express accelerates the application development process with features such as user interface themes, navigational controls, form handlers, and flexible reports. The deployed applications require only a browser and access to an Oracle database running Application Express.

This tutorial is intended to guide you through almost every step of creating an app with a focused amount of context. For more in-depth information about Oracle Application Express, see the "快速入门" section of the Oracle Application Express App Builder User’s Guide.

About This Tutorial

In Oracle Application Express Tutorial: Building an Application, you create a simple project management application for the fictitious AnyCo Corp. The application manages IT project information stored in database tables. This tutorial shows you how to build, modify, and run the application. Each chapter is a further lesson that advances the work of previous one so that the concepts build on one another.

Oracle Application Express Tutorial: Building an Application is intended to be followed in the order it is written.
